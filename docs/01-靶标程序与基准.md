# 靶标程序与基准数据

**环境:**
* 处理器: AMD Ryzen 5 5500U (6 物理核 / 12 逻辑核, 开启 SMT 超线程)
* 操作系统: Linux (Ubuntu 24.04)

---

## 构建靶标程序

编写一个单线程 C 程序，循环执行 10000 次 50x50 矩阵乘法，用 `CLOCK_MONOTONIC` 测量每次耗时。记录平均时间、最坏时间（伪 WCET，10000 次中的最大值）、抖动（最坏减平均）。

### `avionics_target.c`

```c
#define _GNU_SOURCE
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
#include <stdint.h>

#define NUM_ITERATIONS 10000
#define MATRIX_SIZE 50

void simulate_flight_control_logic(float A[MATRIX_SIZE][MATRIX_SIZE],
                                   float B[MATRIX_SIZE][MATRIX_SIZE],
                                   float C[MATRIX_SIZE][MATRIX_SIZE])
{
    for (int i = 0; i < MATRIX_SIZE; i++) {
        for (int j = 0; j < MATRIX_SIZE; j++) {
            C[i][j] = 0.0;
            for (int k = 0; k < MATRIX_SIZE; k++) {
                C[i][j] += A[i][k] * B[k][j];
            }
        }
    }
}

int main() {
    struct timespec start, end;
    uint64_t elapsed_us;
    uint64_t max_time_us = 0;
    uint64_t total_time_us = 0;

    float A[MATRIX_SIZE][MATRIX_SIZE];
    float B[MATRIX_SIZE][MATRIX_SIZE];
    float C[MATRIX_SIZE][MATRIX_SIZE];

    for (int i = 0; i < MATRIX_SIZE; i++) {
        for (int j = 0; j < MATRIX_SIZE; j++) {
            A[i][j] = (float)(rand() % 100) / 10.0;
            B[i][j] = (float)(rand() % 100) / 10.0;
        }
    }

    printf("=> 矩阵大小: %dx%d, 循环次数: %d\n", MATRIX_SIZE, MATRIX_SIZE, NUM_ITERATIONS);

    // 空跑 10 次达到稳态
    for (int i = 0; i < 10; i++)
        simulate_flight_control_logic(A, B, C);

    for (int iter = 0; iter < NUM_ITERATIONS; iter++) {
        clock_gettime(CLOCK_MONOTONIC, &start);
        simulate_flight_control_logic(A, B, C);
        clock_gettime(CLOCK_MONOTONIC, &end);

        elapsed_us = (end.tv_sec - start.tv_sec) * 1000000 +
                     (end.tv_nsec - start.tv_nsec) / 1000;

        if (elapsed_us > max_time_us) max_time_us = elapsed_us;
        total_time_us += elapsed_us;
    }

    uint64_t avg_time_us = total_time_us / NUM_ITERATIONS;
    printf("\n测试完成！\n");
    printf("------------------------------------------------\n");
    printf("平均执行时间 (Average): %llu 微秒 (us)\n", (unsigned long long)avg_time_us);
    printf("最坏情况执行时间 (WCET):  %llu 微秒 (us)\n", (unsigned long long)max_time_us);
    printf("系统抖动 (Jitter):      %llu 微秒 (us)\n", (unsigned long long)(max_time_us - avg_time_us));
    printf("------------------------------------------------\n");
    printf("[防优化输出] C[0][0] = %f\n", C[0][0]);
    return 0;
}
```

编译：

```bash
gcc -O0 avionics_target.c -o avionics_sensor
```

`-O0` 关闭优化，防止编译器把无输出的矩阵运算直接删掉（Dead Code Elimination）。末尾打印 `C[0][0]` 也是同样的目的。

> **说明：** 这里的 WCET 不是严格意义上通过程序分析演算出来的硬上界，而是 10000 次采样中测到的最大值，算一个参考意义上的伪 WCET。

### 基准数据汇总

| 场景 | Avg | WCET | Jitter | 状态分析 |
| :--- | :--- | :--- | :--- | :--- |
| **无干扰** | 462 us | 1016 us | 554 us | 理想状态，缓存命中率高 |
| **强干扰** | 882 us | 10967 us | 10085 us | 发生调度抢占与缓存踩踏 |

后续所有分区优化都围绕一个目标：**在高负载下把 Jitter 从 10085 us 压到百微秒级别**。
